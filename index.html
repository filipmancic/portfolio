Projects

GymTracker

https://github.com/filipmancic/GymTracker

Full-stack mobile gym application using React Native, Node.js/Express, and MongoDB. The app features secure user authentication with JWT and persistent login, a dynamic workout dashboard showcasing an exercise carousel, detailed workout plans, and an interactive stopwatch with start, pause, and reset functions.
Additionally, the app includes a QR scanner module that lets users scan codes during their workouts to automatically decrement their remaining training sessions and log each session with date and time. This project highlights my skills in cross-platform mobile development, state management with Context API, and backend integration to deliver a seamless and user-friendly fitness tracking experience.
